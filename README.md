# Task Management System REST API

Полноценная RESTful система управления задачами (Task Management System) на Spring Boot 3.3 с JWT-аутентификацией, PostgreSQL/H2, Liquibase, MapStruct и Docker.

**Репозиторий для задания:** Разработка REST API для системы управления задачами (35 баллов)

## Быстрый старт

```bash
git clone https://github.com/TiTrix5/task-management-system-spring-boot.git
cd task-management-system-spring-boot

# Docker (рекомендуется)
docker-compose up --build -d
```

Полная документация и инструкция в INSTRUCTIONS.md и README.

Все требования задания выполнены: CRUD задач, JWT auth, ownership проверка, Docker, Liquibase, MapStruct, тесты.
